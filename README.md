# swoft-devtool

Dev tool for swoft

## Install

- composer command

```bash
composer require swoft/devtool
```

## Document

Please see [document site](https://doc.swoft.org)

## Quick use

1. add http middleware in ``

```php

```

2. some optional config for devtool.

```php
'devtool' => [
    'logHttpRequestToConsole' => true,
],
```

## Unit testing

```bash
phpunit test/unit
```
